## 设计理念

本网站采用Claude code + DeepSeek v4 进行vibe coding

站点整体走**瑞士国际主义平面风格**（Swiss International Style），参考了苹果官网和一些独立建筑事务所网站的排版。

核心原则：**少即是多、网格优先、克制用色**。

- 大面积中性灰白底色（`#f5f5f7`），团体色不铺底、不做渐变，仅以一组 2px 细线出现
- 团体卡片采用 `backdrop-blur` 毛玻璃效果，半透明白底 + 极细白色边框 + 微阴影，让卡片轻盈悬浮
- 卡片交替左右偏移排布，间距 36–48 单位，靠不对称的网格节奏而非繁复装饰来构建视觉秩序
- 导航中枢的封面用团体 Logo 原色呈现，不转黑白、不加滤镜，保持各团体品牌辨识度

## 关于羽毛

页面上那枚白色羽毛，是向 LoveLive! 系列「传承之羽」意象的致敬。

在 LoveLive! 的世界里，白色羽毛是贯穿世代的核心符号：μ's 剧场版结尾，一片白羽飘落街头无人拾起——「王位悬空」；Aqours 动画第 12 话，千歌拾起羽毛，完成精神传承；后来理亚获得紫色羽毛，寓意「传承不止一种形态」。

技术上，这枚羽毛不是简单的 CSS 过渡动画，而是用**弹簧-质量-阻尼物理模型**驱动的：
- 每帧计算弹簧力、空气阻力和位移
- 临界阻尼参数保证吸附时不震荡
- 滚动快慢不影响羽毛飘落速度——滚轮停，羽毛就停
- 可以用鼠标拖拽，松手后弹簧拉回最近团体卡片

## 关于技术

本站基于 [Astro](https://astro.build) 构建，样式用 [Tailwind CSS](https://tailwindcss.com)，部署在 [Cloudflare Pages](https://pages.cloudflare.com)。

选 Astro 的原因是它默认零 JS 输出，只在需要交互的地方加客户端脚本——比如羽毛的物理循环和子域名路由切换。博客用 Astro 的内容集合，直接写 Markdown 就能发布。
